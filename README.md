# 🌟 欢迎来到 **NóvaKósmos Worldbuilding Archive**
这里收藏的是 **NóvaKósmos（诺瓦新星地）世界观** 的全部设定文件、角色资料、机构档案，以及来自各个部门的奇奇怪怪（但很努力）的小记录。
这是个正在发芽的原创世界，一切都在成长、在变得更完整，也在慢慢变得更可爱。

## ✨ 这个世界观里，都有什么？
在这个宇宙里，人类的世界与“第二世界”的魔法生态碰撞后，诞生了一个既危险又暖心的故事舞台。

你会看到——

* **N. K. I.（NóvaKósmos Industries）**：一个超越国家、超越人类范式的神秘组织，有着聪明又笨蛋、强大又会闹笑话的成员们。
* **EPÍKLISI、A.E.G.I.S.、PHAROS**：负责调查、战斗、封存、研究的不同部门，各有各的风格，也各有各的惨烈与可爱。
* **Lustyers**：一种新生的人科族群，天真、好奇、能力离谱，而且对朋友特别温柔。
* 偶尔也会出现非常危险的魔法生物、生物百科文档、魔物分类，以及那些让研究员们头疼不已的奇妙生态。

故事整体的气氛是：
**一边努力拯救世界，一边又要担心同事们乱跑、掉进洞穴、把飞机玩坏、或在实验室里吓到自己。**

## 🌼 为什么做这个仓库？
为了整理世界观、方便查阅，也为了让创作中产生的各种文案都能有一个小家。
你看到的所有内容都在不断更新，文件还会增加、结构会变得更清晰，一切都很“WIP中”。
